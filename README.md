# Crab-Keeper

一个使用 Rust 实现的分布式协调服务，设计灵感源自 [Apache ZooKeeper](https://zookeeper.apache.org/)。

## 概览

Crab-Keeper 的开发目标：
- 个人学习 Rust
- 学习 [ITMO 大学分布式系统课程](https://gitlab.com/itmo-compphys/theory-of-distributed-systems) 的理论体系，通过工程（类似 Zookeeper 的服务）实现来验证理论

## 构建与运行

```bash
# 编译
cargo build

# 测试
cargo test

# 运行
cargo run
```
